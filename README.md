## Requirements
```bash
npm -v
# ^5.2.0
```

## Usage
Generate a KORD Guardian micro-service:
```bash
npx kord-guardian generate service
```

Generate a KORD Guardian environment variables config file:
```bash
npx kord-guardian generate env
```

## Contributing

### Setup
```bash
npm install
```

### Build
```bash
npm run build
```

### Test
```bash
npm test
```
